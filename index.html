No JS Web Developer

This site is a coding playground for learning HTML and CSS - no JS allowed!

Hi, I am a web development instructor at Code Immersives career school. My students and I put together this site during instruction hours for the purpose of learning various HTML and CSS features and techniques, including, but not limited to:

  • html5 tags and attributes
  • foreground and background images, sprites
  • media queries and responsive layout
  • dropdown menu and pure CSS responisve menu
  • css3 selectors, properties, transforms, animations

How to use this site:

  • click around and view source
  • inspect using Chrome DevTools' Elements Tab
  • view or download source from github

We intentionally NOT using any css frameworks on the main page of the site. Also the content is gibberish for the most part. Still, I hope this site will provide a useful reference for my students and, purhaps, to someone who begins learning HTML and CSS from scratch.
